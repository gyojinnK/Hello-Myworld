반복문-for
    - 카운터 변수를 기준으로 명령을 여러 번 실행
    - 소스의 양이 줄어 실행 속도가 빨라짐
    1)초기화, 2)조건식, 3)실행문, 4)증감식
    - 싱글루프일 경우 초기화는 한번

while문 : 무한루프를 사용할 때 많이 사용
do-while문 : 사용자에게 의도를 물어보는 프로그램 (최소 한번은 실행될 것이라고 생각되는 프로그램에 사용)

break문 : 반복문의 흐름에서 바로 빠져나올 때 사용
coutinue문 : 주어진 조건에 맞는 값을 만났을 때 실행하던 반복 문장을 건너뛰고 반복문의 맨 앞으로 되돌아감

jQuery란?
    - 웹페이지 제작을 보다 쉽게 만들어주는 자바스크립트 라이브러리.
    - 웹 프로그래밍에서 jQuery는 거의 표준 아닌 표준 라이브러리이기에 좀 더 빨리 익숙하게 하기 위함이다.

jQuery사용법
$()의 정체
    - var $변수 이름 = $("css 선택자")  <- 가장 기초적인 노드 찾는 방법!
    = 함수 이름이 $()인 함수를 호출하는 것.
선택자
    - 말 그대로 css의 선택자를 의미한다.
    - 찾고 싶은 선택자를 만들어 $()함수에 매개변수 값으로 넣어주면 된다.
var $변수 이름
    - $()함수에서 리턴해주는 값을 저장하기 위해서 만든 변수일 뿐이다. 앞에 $를 붙이는 이유는 jQuery기능이 들어있는 변수를 표현하기 위해.
event등록
    - $대상.on("이벤트 이름","이벤트리스너"); 또는 $대상.단축이벤트함수(이벤트리스너);
스타일 설정하기
    - $대상.css("스타일 이름",값);
